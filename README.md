# Work Experience – Harry Potter API Project

This project is a website built with C#. It connects to the [Harry Potter API](https://hp-api.onrender.com/) — a service on the internet that provides data about Harry Potter characters and spells — and displays that data in a web page.

---

## Challenge Setup

You will be working in pairs. One of you will need to **fork this repo** (forking means making your own personal copy of it on GitHub), and then add your partner as a **collaborator** so you can both make changes to it.

To add a collaborator: go to your forked repo on GitHub → Settings → Collaborators → Add people.

You will present on Friday. Both of you will walk through the app and explain what you learnt during the week and how the project works.

---

## Tasks

1. Open the Characters page in the app and have a look at what it shows
2. Try to understand how the app works — it sends a request to an API (a data source on the internet), receives the response as JSON (a text format for storing data), and converts it into something the page can display
3. Build the unfinished Spells page — the Characters page is a good reference for how to do this
4. Stretch goal: fetch data from another Harry Potter API endpoint and display it on a new page

## Prerequisites

You will need to install **Visual Studio 2022 Community Edition**, which is the program used to open, edit, and run this project. It is free.

Download it here: https://visualstudio.microsoft.com/downloads/

During installation, you will be asked what you want to install. Make sure **"ASP.NET and web development"** is ticked.

---

## Getting Started

Once you have forked the repo (see Challenge Setup above), you need to download it onto your computer. This is called **cloning**.

1. Go to your forked repo on GitHub
2. Click the green **Code** button and copy the URL shown
3. Open a terminal — on Windows, search for **Command Prompt** in the Start menu
4. Type the following and press Enter, replacing the URL with the one you copied:
   ```
   git clone https://github.com/your-username/work-experience.git
   ```
5. This will create a folder called `work-experience` on your computer. Open it
6. Inside that folder, open the `HarryPotter/` folder
7. Double-click `HarryPotter.sln` — this will open the project in Visual Studio
8. Click the green **`https`** run button at the top of Visual Studio to start the app
9. A browser window will open showing the Harry Potter API Project homepage
